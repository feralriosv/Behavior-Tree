# Behavior Tree Simulator

Java implementation of a behavior tree interpreter controlling multiple agents (ladybugs) in a 2D grid environment.

Agents execute decision logic defined by behavior trees and interact with the environment through actions such as moving, turning, collecting leaves, and flying to specific positions.

## Features

- Behavior tree interpreter
- Support for sequence, fallback, parallel, condition, and action nodes
- Multi-agent simulation
- 2D grid environment
- Command-line interface
- Loading of boards and behavior trees from files

## Example Commands

load board ./board.txt  
load trees ./tree.txt  
list ladybugs  
print position 1  
next action

## Technologies

Java  
Object-Oriented Programming  
Tree data structures  
Command-line interfaces

## Purpose

This project was developed as part of a university assignment to explore behavior trees, a decision-making architecture commonly used in game AI and robotics.
